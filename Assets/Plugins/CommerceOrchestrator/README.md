# Commerce Orchestrator: Standalone IAP & LiveOps Pricing Sync

Commerce Orchestrator — це універсальний, повністю автономний Unity-плагін для інтеграції внутрішньоігрових покупок (IAP) маркетів Google Play та App Store з можливістю динамічного керування цінами та нагородами через Google Sheets (LiveOps).

## Основні переваги
1. **Повна автономність (Standalone)** — плагін розроблено без жорстких прив'язок до конкретних DI-контейнерів (Zenject чи VContainer) та ігрових менеджерів інвентарю.
2. **Динамічний LiveOps (Google Sheets Sync)** — керуйте тарифами, знижками та наповненням товарів у реальному часі прямо з вашої хмарної таблиці без випуску оновлень додатку.
3. **Авто-парсер нагород (Reward Auto-parser)** — автоматичний інтелектуальний розбір рядків нагород на кшталт `"100 Quants + 50 Horas"` на структуровані сутності.
4. **Готовність до підписок (Subscription Ready)** — повноцінна підтримка не лише разових продуктів, а й підписок (наприклад, підписка "Enlightened").
5. **Надійне кешування та фоллбеки** — потрійний рівень стабільності: завантаження з хмари → локальний JSON-кеш `PlayerPrefs` → локальний ScriptableObject `StoreDatabase`.
6. **Сучасний візуальний UX** — вбудована анімація розгортання карток товарів із затемненням фону на основі DOTween.

## Інтеграція
Для початку роботи створіть ScriptableObject конфігурації магазину (`StoreDatabase`) через меню `Create -> Commerce Orchestrator -> Store Database`. Зареєструйте товари та підключіть провайдер цін (локальний чи Google Sheets) до менеджера `StoreManager`.
